Project 1 CSE Simulator

A VR game that simulates the life of a CSE student in the old 1695 BBB. Created using Unreal Engine 5.2.1 to develop the VR gameplay. GitHub was used for version control and collaboration with Milo Baumhardt. Jira was used for managing sprint tasks. Models sourced from Sketchfab and audio sourced from freesound.org. Animations created using Mixamo and all development was tested on a Meta Quest Pro through the UMich VizStudio. CLICK THE IMAGE TO VIEW THE DEMO ON YOUTUBE!
